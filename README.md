# React Native + Taildwind + Typescript + OneSignal

## Base:
This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Used libraries:
- nativewind / tailwindcss
- typescript
- signal-expo-plugin
- others

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
